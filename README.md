# Data-Utility-

This is a Data Structure of all the monsters in DND. This prints out a monster if you are using any getBy funictions that are available in Monsters.java. There is also the getArmor, and getByArmor. to sort them by armor. For the getByCR and getLostestArmor, you need to go gto the Main class and change it to match the the Challenge rating and Armor class you are.
